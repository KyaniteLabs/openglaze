## How to Use This Document

### Layering Convention

**"A OVER B"** means:
- **A** = TOP layer (applied 2nd, on top)
- **B** = BASE layer (applied 1st, on bisque)

Example: "Clear OVER Red" = Dip Red first → Dry → Dip Clear second

### Application Methods

| Method | Description | Best For |
|--------|-------------|----------|
| Double-dip | Dip in glaze A, dry, dip in glaze B | Even coverage, production pieces |
| Brush-over | Brush glaze A, dry, brush glaze B | Controlled areas, decorative effects |
| Spray-over | Spray glaze A, dry, spray glaze B | Gradual transitions, large pieces |

## Basic Glaze Chemistry (Applies to All Temperatures)

### The Unity Molecular Formula (UMF)

Glazes are balanced by molecular ratios, not weight:
- **Fluxes** (KNaO, CaO, MgO, ZnO, etc.) - melt the glaze
- **Alumina** (Al₂O₃) - stabilizes, controls melt viscosity
- **Silica** (SiO₂) - glass former, controls durability

### Key Ratios

| Ratio | Effect |
|-------|--------|
| **Silica:Alumina** | Higher = more glossy, lower = more matte |
| **Flux ratio** | Determines melting temperature |
| **Boron content** | Lower melting point, often used in mid-fire |

### Colorants

| Colorant | Oxidation | Reduction |
|----------|-----------|-----------|
| **Iron** | Amber, brown, rust | Celadon green, tenmoku black |
| **Copper** | Green, turquoise | Red (if heavy reduction) |
| **Cobalt** | Blue | Blue (stable) |
| **Chrome** | Green | May interact with tin for pink |
| **Rutile** | Tan, gold specks | Variegated, mottled |

## Mid-Fire Specifics (Cone 5-6)

### Temperature Range

| Cone | Approximate Temperature |
|------|------------------------|
| Cone 5 | ~2167°F / 1186°C |
| Cone 6 | ~2232°F / 1222°C |

### Atmosphere Differences from Cone 10

| Factor | Cone 5-6 Oxidation | Cone 10 Reduction |
|--------|-------------------|-------------------|
| **Iron response** | Browns, ambers | Greens, blacks |
| **Copper response** | Greens, turquoise | Reds possible |
| **Carbon trap** | Not possible | Shinos, lusters |
| **Surface depth** | Bright, clean | Deep, variegated |

### Common Mid-Fire Base Types

| Base Type | Characteristics |
|-----------|-----------------|
| **Boron-based** | Lower melt, glossy |
| **Strontium matte** | Smooth matte, reliable |
| **Zinc crystalline** | Crystal growth possible |
| **Nepheline syenite** | Lower melting feldspar |

## Appendix: Test Protocol

### Before Testing Any Combination

1. **Check reference** - Is it documented?
2. **Assess risk** - Fluidity and opacity compatibility?
3. **Plan application** - Which method? Which order?
4. **Document** - Record exact process

### Test Tile Method

1. Apply BASE glaze to half the tile
2. Apply TOP glaze to entire tile (creates overlap zone)
3. Mark application order and thickness
4. Fire with standard Cone 5-6 schedule
5. Evaluate all three zones: BASE only, TOP only, OVERLAP
