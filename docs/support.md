# Supporting OpenGlaze

OpenGlaze is **free, open source, and self-hosted**. The entire project — UMF calculator, recipe optimizer, chemistry engine, and all features — is available under the MIT license at no cost.

If OpenGlaze saves you a test tile, a failed kiln load, or hours of recipe tweaking, you can say thank you by supporting the project.

## What Your Support Covers

- **Development time** — New features, bug fixes, optimizer improvements
- **Testing** — Test tiles, material analysis, firing validation
- **Documentation** — User guides, API docs, tutorials
- **Community** — Answering questions, reviewing contributions

## What You Get

**Nothing gated.** The entire tool is free regardless of whether you contribute. Supporters may get:

- A mention in the README or release notes
- Early access to new features (via the public GitHub repo, same as everyone)
- Gratitude from the creator and the community

## No Pressure

OpenGlaze exists because the creator needed it and wanted to share it. Use it, fork it, modify it — no obligation to pay. If you do choose to support the project, it is deeply appreciated.

---

*OpenGlaze follows the Glazy model: the tool is unconditionally free. Support is voluntary gratitude, not payment for access.*
