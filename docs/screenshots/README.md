# Screenshots

Real screenshots from the live app at https://openglaze.kyanitelabs.tech should be placed here.

## Required Screenshots

| File | What to Capture | Size |
|---|---|---|
| `glaze-calculator.png` | UMF calculator with a loaded recipe showing oxide breakdown and SiO2:Al2O3 ratio | 1280x720 min |
| `glaze-library.png` | Glaze library grid showing multiple glazes with color swatches | 1280x720 min |
| `recipe-optimizer.png` | Recipe optimizer showing a before/after suggestion | 1280x720 min |
| `ai-assistant.png` | Kama assistant answering a glaze chemistry question | 1280x720 min |
| `mobile-view.png` | Responsive view on mobile viewport | 390x844 min |
| `demo.gif` | 15-30 second walkthrough: add recipe -> UMF calc -> optimizer -> CTE analysis | 1280x720 |

## Capture Instructions

1. Open https://openglaze.kyanitelabs.tech in Chrome
2. Use DevTools device toolbar (Cmd+Shift+M) for mobile screenshots
3. For GIF, use a screen recorder (Loom, Kap, or ffmpeg)
4. Save as PNG (screenshots) or GIF (demo)
5. Optimize: `brew install imagemagick && mogrify -resize 1280x720\> *.png`
