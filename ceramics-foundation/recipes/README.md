# Glaze Recipes

Collection of documented glaze recipes organized by source.

## Structure

```
recipes/
├── laguna/      # Laguna Clay Company dry glazes
├── aardvark/    # Aardvark Clay & Supply glazes
└── studio/      # Custom and community recipes
```

## Recipe Format

All recipes follow this YAML structure:

```yaml
name: Recipe Name
source: Origin (catalog, book, community)
cone: "10"
atmosphere: reduction  # or oxidation, both
recipe:
  - material: Material Name
    amount: 40  # percentage
  - material: Second Material
    amount: 25
color: Color description
surface: gloss | matt | semi-matt
notes: Additional notes
food_safe: true | false | unknown
tested: true | false
```
