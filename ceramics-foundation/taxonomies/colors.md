## Usage Guidelines

1. **Be Specific First** - Use precise terms (e.g., "celadon" not "green")
2. **Fallback When Needed** - Use generic colors only if specific identification isn't possible
3. **Combine Terms** - Use hyphenated combos for accuracy (e.g., "blue-gray", "red-orange")
4. **Consider Context** - Glaze color may differ from raw clay color
5. **Firing Effects** - Note that reduction/oxidation changes color outcomes

## Firing Context

All colors described in context of **cone 10 reduction firing**, which affects:
- Iron-rich clays develop darker, reduced colors
- Glazes break differently at edges
- Carbon trapping creates dark patches
- Copper flashes produce red/pink highlights
